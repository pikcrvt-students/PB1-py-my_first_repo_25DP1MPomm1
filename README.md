## Uzņēmums

### mikrotik

![mikrotik logo](#)

- Dibināts 1996. Gadā, Latvijā.
- Uzņēmums izstrādā un ražo MikroTik RouterOS programmatūru un RouterBOARD maršrutētājus.
- Uzņēmuma mērķis ir nodrošināt jaudīgus un ērti lietojamus datortīklu pārvaldes rīkus visplašākajam lietotāju lokam.
- Mikrotīkls aktīvi iesaistās zinātnes, medicīnas un izglītības nozaru atbalstā. [^1.]
- Uzņēmuma vērtība 2025. Gadā - 1,18 miljards € [^3.]
