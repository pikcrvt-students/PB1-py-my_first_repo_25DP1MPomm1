# Datorspēle - akmens šķēres papīrīts

## Spēles mērķis: izvēlēties opciju, kas pieviec pretinieka izvēli.

* Akmens pieveic šķēres.
* Šķēres pieveic papīru.
* Papīrs pieveic akmeni.

*Spēlētāju skaits: 2 vai vairāk*

## Spēles gaita:

1. Spēlētāji izvēles vienu no opcijāsm: Akmens, Šēres, Papīrs.
2. Spēlētāji salīdzina izvēlētas opcijas un izlēmj kurš uzvarēja.
